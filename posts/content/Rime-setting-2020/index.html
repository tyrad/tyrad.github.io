鼠须管(Rime)使用配置(2020)

June 25, 2020
软件
Rime • 软件

总结下个人使用鼠须管的一些配置,本文主要以记录为主。

当前的电脑环境为 macos 10.15。

布局设置为横向

点击图标 -> 用户设定,可以打开用户配置目录,编辑squirrel.custom.yaml文件，

patch:
  style:
    horizontal: true # 候选窗横向显示

切换为简体中文

输入法安装完成后，使用 ctrol+` 可以进行语言的切换。

笔者使用的是朙月拼音—简化字,用户可以根据实际需求进行自由选择。

设置语言切换时(shift),字符直接进行清空并显示

搜狗输入法在点击 shift 切换时，会直接清空输入的英文内容，并默认填充到文本里。 rime 也支持这样的设置。

参考此处的做法,修改default.custom.yaml文件,修改内容如下:

# default.custom.yaml, 全局生效
patch:
  "menu/page_size": 5 # 每页的候选项目的个数,范围为1-9 数据手册
  ascii_composer/good_old_caps_lock: true
  # 配置当语言切换时，字符直接进行清空并显示。参考: https://zhuanlan.zhihu.com/p/91129641
  ascii_composer/switch_key:
    Caps_Lock: clear
    Shift_L: commit_code
    Shift_R: inline_ascii
    Control_L: noop
    Control_R: noop

扩充词库

参考Rime 输入法—鼠须管(Squirrel)词库添加及配置 文章的做法，

  1. 下载朙月拼音擴充詞庫，下载解压，放到 rime 目录里去（有冲突文件的话需要留意下，按需修改文件）

  2. 注意luna_pinyin.custom.yaml,可以查看里面的注释。由于笔者使用的“朙月拼音·简化字”对应luna_pinyin_simp,故将luna_pinyin.custom.yaml文件名改为了luna_pinyin_simp.custom.yaml

  3. 点击图标，选择重新部署

使用搜狗输入法词库

近期在 github 上发现了一个 python 脚本: scel2txt,可以搜狗词库转为我们需要的格式。

1). 根据 scel2txt 项目的说明生成luna_pinyin.sogou.dict.yaml文件（可以看到配置文件中 name 为luna_pinyin.sogou），并将该文件添加到配置目录中。

2). 编辑luna_pinyin.extended.dict.yaml（步骤扩充词汇时生成）,添加行- luna_pinyin.sogou,如下:

---
#....
import_tables:
  - luna_pinyin
  - luna_pinyin.hanyu
  - luna_pinyin.poetry
  - luna_pinyin.cn_en
  - luna_pinyin.sogou #添加该行
#...

输入法切换时，默认显示中文数据法

例如设置简体拼音，修改修改文件luna_pinyin_simp.custom.yaml

patch:
  switches/@0/reset: 0

主题配色设置

可参考: https://mritd.me/2019/03/23/oh-my-rime/

同步

修改installation.yaml

# 修改id，同步时会在同步目录下生成文件夹，文件夹名对应id
installation_id: "rime-mac-sync"
# 指定同步目录，例如icloud的目录
sync_dir: "/Users/panda/Library/Mobile Documents/com~apple~CloudDocs/Rime"

点击同步用户数据,即可同步到指定目录。
